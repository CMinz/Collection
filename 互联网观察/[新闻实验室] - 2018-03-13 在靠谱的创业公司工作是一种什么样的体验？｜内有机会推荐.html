【实验室主任按】VUE是我很喜欢的一款短视频拍摄剪辑app，也曾听说在那里工作是不错的体验。现在，VUE团队专门通过新闻实验室发出如下招聘信息，室友们抓住机会～

前段时间，网络上对于年轻人应该去创业公司还是大公司工作有过一波讨论。

 

大公司能提供更大的平台，有更规范的制度和员工培训体系，但是对于个人来说，能够发挥的空间有限。在小公司工作不确定性更大，但是如果在合适的时机加入对的团队，能力和薪资的发展速度都会快于同龄人。

 

比如这家创业公司，你可能没有听过他们的公司名，但是一定用过或者看见过他们的产品：VUE app，一款设计精美的短视频拍摄工具。

VUE 的 90 后产品设计师，本科毕业第二年，就独立负责了一款全新 APP 的整体设计工作。她创作的 LOGO在产品上线后的第一年内就出现在了超过 1000 万用户的手机屏幕上，她优化的产品界面累计帮助了数千万人拍出记录生活，分享心情的短视频。

在 VUE 这样的创业公司，只要你有能力，舞台足够大。

为什么你该来这样的创业公司？


● 产品本身足够优秀，公司发展好

    ○ VUE 连续两年作为苹果 App Store 年度精选应用被推荐

    ○ 产品快速增长，融资顺利，过往投资方包括真格、贝塔斯曼等



● 福利好，待遇佳

    ○ 提供与大公司相比也有竞争力的薪水，年底有双薪加奖金，视工作表现一年还有两次涨薪机会

    ○ 公司过去两年曾集体去马尔代夫和塞舌尔团建，包括实习生

    ○ 工作日有餐补，免费零食和水果


2016年 VUE 马尔代夫团建，当时团队还只有 10 个人左右


● 公司文化包容开放

    ○ 在 VUE，我们看重个人成长，尊重个人价值的实现，包容多元的文化观念

    ○ 你会遇到专业能力一流，同时有着丰富爱好的同事，电影、ACG、古风、鹦鹉等

 

● 团队扁平，使用硅谷风格的管理模式

    ○ 创始团队来自腾讯、微软、网易、豌豆荚等明星公司

    ○ 团队使用能提高效率的办公软件

    ○ 自由工作时间，上下班不打卡，不提倡加班


VUE 办公室前厅


来这里工作吧


这次专门向新闻实验室的读者开放的是 VUE 市场部的两份工作。具体职责和要求如下：

 

新媒体运营

 

职位描述：

1. 负责新媒体平台传播内容的策划、撰写和运营

2. 研究网络热点话题，实时更新自媒体平台内容

3. 负责线上/线下专题内容、专题活动

4. 数据监测、定期进行活动评估和总结报告

5. 分析用户需求、收集用户反馈


任职要求：

1. 熟悉 Social Media，并拥有多个平台的运营推广经验

2. 对文字有深度理解和把握，能独立写稿，运营内容专题

3. 了解视频拍摄相关知识

4. 熟练使用图片处理或视频剪辑工具


商务拓展

 

职位描述：

1. 与合作方进行沟通谈判，拓展合作关系

2. 参与拟定产品的推广方案，并执行落地

3. 整合资源，完成异业合作的资源置换

4. 分析业务数据，完成产品用户增长的目标


任职要求：

1. 对互联网行业和短视频领域有一定的经验或了解

2. 擅长与人沟通谈判，有较强的逻辑思维能力

3. 目标导向，有责任感和上进心


以上职位均招聘全职，工作地点在北京东三环，可以发送简历和应聘职位至 recruit@vue.video


如果有北京的在校学生，对 VUE 感兴趣的话，也可以来邮件沟通。实习生在 VUE 不需要帮忙贴发票，而是一样会面对挑战性的工作。
